# 上报用户地理位置

## 接口
```
/users/{user}/geohash
```

## 请求方式
```
PATCH
```

## 接口变量
| name     | desc     |
|----------|:--------:|
| user     | 用户id   |

## 请求体
| name     | type     | must     | desc     |
|----------|:--------:|:--------:|:--------:|
| latitude | float    | yes      | 纬度      |
| longitude| float    | yes      | 经度      |

## 返回数据
```json5
{
  "status": true,
  "message": "更新成功",
  "data": null
}
```
